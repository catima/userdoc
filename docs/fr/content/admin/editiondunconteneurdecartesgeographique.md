- **Etape 1** - <a href="#ajoutpage">Ajouter une page</a> (étape initiale)
- **Etape 2** - <a href="#editionpage">Editer la page</a> (étape précédente)
- **Etape 3** - Ajouter un conteneur MAP (détails ici)
