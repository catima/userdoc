**Options d'export**

Il y a 3 options disponibles dans l'export au format CSV:
1. Inclure *les fichiers* si vous avez besoin de ces données

Les options ci-après sont disponibles uniquement dans l'export CSV:
- L'*identifiant Catima* (point 2 ci-dessous) est utile notamment si vous envisagez d'importer des données en masse avec la fonctionnalité d'import de Catima 
- l'inclusion des *slugs* (point 3 ci-dessous) est utile pour l'import de données en masse

![Options CSV AND](assets/export/export-options-csv.png) 


**Résultats**

Les informations exportées sont les suivantes :

1.	La **structure des données** n'est pas formellement décrite dans cet export, mais se cache derrière les données des fichiers \*.csv
2.	Les **données** du catalogue sont sauvegardées dans les fichiers \*.csv (note: il y a 1 fichier csv pour chaque type de fiche)
3.	(optionnel) les *fichiers* et les *images* sont dans le dossier **files**
4.  Les **métadonnées de l'export** (date et heure de l'export) sont dans le fichier *meta.json*
