Par défaut, CATIMA génère un affichage par type de fiche en créant une page dédiée qui n'est pas modifiable.  

Ainsi pour changer le style d'affichage des contenus (grille d'images, grille de texte ou liste) ou pour changer les champs affichés, la création d'une nouvelle page est nécessaire. 

> NB : le nom utilisé pour l'affichage des fiches (par défaut le nom donné au *type de fiche* est affiché) peut être changé sans la création d'une nouvelle page. Pour cela il s'agit de passer par l'*"*Affichage des fiches*"* dans la section de configuration du *type de fiche*, c'est-à-dire sous la liste des champs du *type de fiche*.

Les *pages* peuvent contenir du texte mis en forme, du contenu (c'est-à-dire des champs) issu des types de fiches ou une carte géographique par exemple, mais également, pour les utilisateurs avancés, du code HTML et Markdown.
