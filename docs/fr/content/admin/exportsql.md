**Options d'export**

Les options disponibles pour l'export au format SQL sont les mêmes que pour l'export catima:

![Options SQL AND](assets/export/export-options-catima-sql.png) 


**Résultat** 

Toutefois, les fichiers exportés sont moins nombreux et les informations moins complètes que pour le format JSON. 
L'export SQL comprend:

1.	La **structure** des données ...
2.	... et les **données** du catalogue sont tous deux dans le fichier avec l'extension \*.sql (ici: *mes-donnees-sb.sql*)
3.	(optionnel) les *fichiers* et les *images* sont dans le dossier **files**
4.  Les **métadonnées de l'export** (date et heure de l'export) sont dans le fichier *meta.json*
