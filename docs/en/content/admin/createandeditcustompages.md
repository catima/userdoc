CATIMA has a default mode to display the item to the final user. This mode can not be modified, but we can create **pages** where the layout can be modified as needed.

Thus, in order to amend the display of items (grid, thumbnails or timeline) or to modify the fields to be displayed, the creation of a new page is needed.

> Note : the name used in the Item view page (name of the item-type) can be changed without creating a new page. To do that, select the item type and click *New item view* in the configuration page of that item type (this section shows at the bottom of the page).

Pages can contain formatted text, content coming from item types (such as specific fields), a geographic map, but also HTML and markup code as useful for more advanced users.
