**Options**

3 options are available in the CSV export:
1. Include *files* if you need this data

Then the below options are only available in the CSV export:
- *Catima identifier* (point 2 below) is useful if you plan to import data in batch via Catima's embedded import functionnality 
- *Slugs* (point 3 below) are useful for batch data import

**Results**

Information available in this export is detailed hereafter :

1.	The **data structure** is not formally described in the CSV export, but can be more or less guessed from the data present in the \*.csv files.
2.	The **data** is saved in the \*.csv files (note: there is 1 csv file per item-type)
3.	(optionnal) the *files* and *images* are in folder **files**
4.  The **export metadata** (i.e. date and time of the export) is in file *meta.json*
