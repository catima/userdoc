If you cannot find information about a specific feature on this page, look in the documentation for a role with fewer rights. CATIMA feature documentation is organized as follows:

- **Search and navigation** - these features are described in the help pages related to <a href="./guest">Visitor</a> or <a href="./user">User</a> role
- **DATA** - features related to the DATA menu can be found in the help section for <a href="./editors">Editor</a>, <a href="./supereditors">Super Editor</a>, or <a href="./reviewer">Supervisor</a> roles
- **SETUP** - features available in the SETUP menu are described in the <a href="./admin">Admin</a> role documentation pages
