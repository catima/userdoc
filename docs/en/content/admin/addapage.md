Adding a page with new content requires three distinct steps:
- **Step 1** - Add a page (détails are provide in this section of the help)
- **Step 2** - <a href="#addacontent">Edit a page</a> to start adding content (middle step)
- **Step 3** - Add one or more containers, for instance <a href="#map-container">a MAP container</a> (final step)

To add a page, select the SETUP mode and click on the "Pages" menu in the left navigation of your catalog. 

The list of existing pages is displayed. 

> Note : initially, there will be none.

![](assets/pages/page.png)

Click on "New page". 

The following parameters will show:

 ![](assets/pages/new_page.png) 

Chose the page title(s) in the languages of the catalog and chose a simplified name for the page that is called "slug" and that will be part of the url of the page that is generated by CATIMA. The slug must be unique and should contain only numbers, hyphens or letters (without accents). Slugs are usually in English even if the primary language of the catalog is another idiom.

> Example of slug : "home", "library", "contact-page"

Once all fields are completed, you can save "*Create page*" (this will bring you back to the *Setup* menu) or you can click "*Create and add another*", which will save that page and allows you to create another page right away.
